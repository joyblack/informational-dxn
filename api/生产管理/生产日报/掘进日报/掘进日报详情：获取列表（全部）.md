# 简介
获取列表。

# 访问地址
produce-driving-daily-detail/getAllList

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|drivingFaceId|[number]||掘进工作面信息ID|
|dailyTimeStart|[date]||日报起始时间|
|dailyTimeEnd|[date]||日报截止时间|
|shifts|[enum]||班次：参考参数参考表 **4.7、掘进日报(driving-daily)**|
|drillTeamId|[number]||队伍ID，其实就是部门ID|

## 请求示例
```json
{
	"drivingTeamId": 1
}
```
