# 简介

# 访问地址
produce-drill-daily/getMonthFillStatus

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|time|[date]|是|日期，系统会自动提取该日期所属月份。|

## 请求示例
```json
{
	"time": "2019-09-12"
}
```
