# 返回结果
**成功**
```json
{
    "state": true,
    "message": "操作成功",
    "detailMessage": "",
    "data": [
        {
            "workName": "工作面A",
            "totalLength": 2800.50,
            "doneLength": 200.40,
            "progress": "0.07",
            "progressUsePercent": "7%",
            "leftLength": 2600.10
        },
        {
            "workName": "工作面B",
            "totalLength": 2800.50,
            "doneLength": 200.40,
            "progress": "0.07",
            "progressUsePercent": "7%",
            "leftLength": 2600.10
        }
    ],
    "code": 200
}
```

**返回数据解析**

|参数名|类型|说明|
|-|-|-|
|workName|[string]|工作名称|
|totalLength|[number]|总长度|
|doneLength|[number]|已完成长度|
|progress|[string]|完成进度，小数表示，保留两位有效小数位。|
|progressUsePercent|[string]|完成进度，百分数表示|
