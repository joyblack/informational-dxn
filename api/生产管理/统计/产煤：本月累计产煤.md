# 返回结果
**成功**
```json
{
    "state": true,
    "message": "操作成功",
    "detailMessage": "",
    "data": {
        "totalOutput": 0,
        "drivingOutput": 0,
        "miningOutput": 0
    },
    "code": 200
}
```

**返回数据解析**

|参数名|类型|说明|
|-|-|-|
|totalOutput|[decimal]|本月累计产煤|
|drivingOutput|[decimal]|掘进产煤|
|miningOutput|[decimal]|回采产煤|
