# 简介
统计月度数据，需要注意的是，由于前年没有数据信息，因此在此处我们不展示同比数据（红色折线），因此只需展示堆叠柱状图即可。

# 访问地址
produce-statistic/getEveryMonth

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
无

## 请求示例
无
