# 简介

# 访问地址
cm-platform/getPagerList

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|name|string|否|煤矿名称的模糊查询搜索串|
|loginName|string|否|登录名称的模糊查询搜索串|

**其他参数参考《参数参考表》中分页参数模块。**

## 请求示例
```json
{
	"cmName": "1",
	"loginName": "zhaoyi",
    "page": 1,
	"size": 3,
	"sort": "createTime",
	"order": "DESC"
}
```
