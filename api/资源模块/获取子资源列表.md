# 简介
根据父部门ID获取子部门列表。顶层部门可传递参数0。

# 访问地址
server:port/information-dxn/v1/resource/getChildren

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|parentId|[number]|是|父节点的ID|

## 请求示例
```json
{
	"parentId": 0
}
```
