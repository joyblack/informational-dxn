# 简介
获取职位列表，分页查询。

# 访问地址
position/getPagerList

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|search|string|否|通用查询子串，可对职位名称进行模糊查询。|

* 其他参数参考 **参数参考表：分页通用参数**


## 请求示例
```json
{
	"search": "",
	"size": 5,
	"page": 1,
	"sort": "createTime",
	"order": "ASC"
}
```
