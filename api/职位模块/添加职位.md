# 简介

# 访问地址
position/add

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|postionName|[string]|是|名称|
|describes|[string]||描述信息|
|remarks|[string]||备注信息|

## 请求示例
```json
{
	"positionName": "研发工程师",
	"describes": "是一个吃青春饭的工作。",
	"remarks": "有前途吗？"
}
```

# 返回结果
**成功**
```json
{
    "state": true,
    "message": "操作成功",
    "detailMessage": "",
    "data": {
        "id": 101,
        "isDelete": false,
        "createTime": "2019-08-17 14:08:15",
        "updateTime": "2019-08-17 14:08:15",
        "remarks": "有前途吗？",
        "positionName": "研发工程师",
        "describes": "是一个吃青春饭的工作。"
    },
    "code": 200
}
```

**失败**
```json
{
    "state": false,
    "message": "职位名称已存在",
    "detailMessage": "职位名称已存在",
    "data": null,
    "code": 104001
}
```
