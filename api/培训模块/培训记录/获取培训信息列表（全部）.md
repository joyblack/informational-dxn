# 简介
获取职位列表，分页查询。

# 访问地址
train-training/getAllList

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|说明|
|-|-|-|-|
|trainingName|[string]||名称|
|trainingTimeStart|[date]||培训日期开始时间|
|trainingTimeEnd|[date]||培训日期截止时间|
|trainingUsername|[string]||培训人|
|departmentId|[number||受训部门ID|
|trainingContent|[string]||培训内容|

## 请求示例
```json
{
	"trainingName": "海边",
}
```
