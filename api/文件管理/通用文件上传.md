# 简介
该接口可提供通用的文件上传模块信息。

# 访问地址
file/upload

# 请求参数

## 请求方式
POST

## 请求格式
form-data

## 请求数据
|字段|类型|值|
|-|-|-|
|file|[file]|文件|
|module|[enum]|所属模块，支持的值请参考**参数参考表**|
