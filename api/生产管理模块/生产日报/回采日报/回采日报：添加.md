# 简介

# 访问地址
produce-back-mining-daily/add

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|-|
|backMiningFaceId|[number]|是|日报对应的回采工作面信息ID|
|dailyTime|[date]|日报的填写日期：yyyy-MM-dd|

## 请求示例
```json
{
	"backMiningFaceId":1,
	"dailyTime": "2019-08-24"
}
```
