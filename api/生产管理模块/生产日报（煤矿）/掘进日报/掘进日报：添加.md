# 简介

# 访问地址
produce-driving-daily/add

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|-|
|drillFaceId|[number]|是|日报对应的掘进工作面面信息ID|
|dailyTime|[date]|日报的填写日期：yyyy-MM-dd|

## 请求示例
```json
{
	"drivingFaceId":1,
	"dailyTime": "2019-08-24"
}
```
