# 简介
获取列表，分页查询。

# 访问地址
produce-drill-daily-detail/getPagerList

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|drillWorkName|[string]||名称|
|dailyTimeStart|[date]||日报起始时间|
|dailyTimeEnd|[date]||日报截止时间|
|shifts|[enum]||班次：参考参数参考表 **4.5、钻孔日报(drill-daily)**|
|drillTeamId|[number]||打钻队伍ID，其实就是部门ID|

## 请求示例
```json
{
	"drillWorkName": "钻孔工作"
}
```
