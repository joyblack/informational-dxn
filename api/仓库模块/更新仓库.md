# 简介

# 访问地址
server:port/information-dxn/v1/storehouse/update

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|id|[number]|是|ID|
|code|[string]||仓库编码|
|responseUser|[string]||负责人信息|
|admin|[string]||管理员|
|status|[number]||0-启用 1-停用|
## 请求示例
```json
{
	"id": 7,
    "code": "123456",
    "responseUser": "赵义",
    "name": "3号仓库",
    "admin": "wo",
    "status": 0
}
```

# 返回结果
**成功**
```json
{
    "state": true,
    "message": "操作成功",
    "detailMessage": "",
    "data": {
        "id": 7,
        "isDelete": false,
        "createTime": "2019-08-13 17:51:10",
        "updateTime": "2019-08-13 17:51:10",
        "remark": null,
        "status": 0,
        "name": "3号仓库",
        "responseUser": "赵义",
        "code": "123456",
        "admin": "wo"
    },
    "code": 200
}
```

**失败**
```json
{
    "state": false,
    "message": "仓库名称已存在",
    "detailMessage": "仓库名称已存在",
    "data": null,
    "code": 104001
}
```
