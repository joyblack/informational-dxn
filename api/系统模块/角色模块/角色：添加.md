# 简介

# 访问地址
system-role/add

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|roleName|[string]|是|角色名称|
|permissions|[string]||若不填，代表该角色拥有所有权限，等价于全勾选。|
## 请求示例
```json
{
    "roleName": "资料管理员",
    "permissions": "1-2-3-10-15-25"
}
```
