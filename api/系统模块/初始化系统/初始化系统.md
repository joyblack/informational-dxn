# 简介
系统启动时，需要为系统添加：
* 配置信息：添加默认密码配置项；
* 部门信息：添加集团部门；
* 用户信息：添加默认超管账户、超管密码，属于集团；

# 访问地址
system/start

# 请求参数

## 请求方式
POST

## 请求格式

## 请求数据
无
