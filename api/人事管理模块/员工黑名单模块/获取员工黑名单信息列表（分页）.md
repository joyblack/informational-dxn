# 简介
获取全部列表。

# 访问地址
staff-blacklist/getPagerList

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|username|[string]||查询关键字：姓名|
|idNumber|[string]||查询关键字：身份证|
|phone|[string]||查询关键字：联系方式（电话号码）|
|blacklistReasons|[string]||查询关键字：原因|
|overTimeStart|[date]||解禁时间的起始时间|
|overTimeEnd|[date]||解禁时间的截止时间|

* 其他参数参考 **参数参考表：分页通用参数**

## 请求示例
```json
{
	"leaveType": "NORMAL"
}
```
