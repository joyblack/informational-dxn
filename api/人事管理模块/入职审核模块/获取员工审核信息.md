# 简介
进行审核时，需要知道员工被审核的原因，可通过此接口获取。

# 访问地址
server:port/information-dxn/v1/staff-review/getReviewInfo

# 请求参数

## 请求方式
POST

## 请求格式
JSON

## 请求数据
|参数名|类型|必填|说明|
|-|-|-|-|
|id|[number]|是|员工ID|


## 请求示例
```json
{
    "state": true,
    "message": "操作成功",
    "detailMessage": "",
    "data": {
        "id": 1,
        "isDelete": false,
        "createTime": "2019-08-16 14:08:56",
        "updateTime": "2019-08-16 14:08:56",
        "status": null,
        "remark": null,
        "staffId": 5,
        "reviewReasons": "已在煤矿平台入职，职务为业务员3\r\n",
        "reviewStatus": 0,
        "reviewTime": null,
        "reviewUserId": 0
    },
    "code": 200
}
```
